# 🔩 Steel Surface Defect Classification

Deep Learning project for automatic defect detection on steel surfaces.

## Dataset

NEU Surface Defect Database — 1800 images, 6 classes, perfectly balanced.

## Approach

- Model A: Custom CNN (from scratch)
- Model B: Transfer Learning with MobileNetV2

## Tech Stack

Python · TensorFlow/Keras · Google Colab (GPU T4)
